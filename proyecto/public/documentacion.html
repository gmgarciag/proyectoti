Función: Entregar id Grupo

Retorna el id del grupo

- URL: integra1.ing.puc.cl/api/ids/grupo

- Método: GET

- Retorno: {"id":string}

- Método: GET

- Retorno: {"id":string}

- Retorno: {"id":string}

Función: Consultar Stock

Esta función tiene como objetivo consulta el stock de un producto en nuestra empresa,
recibe como parametro el sku de un producto y retorna un JSON
con su stock y su sku.

- URL: integra1.ing.puc.cl/api/consultar/:sku

- Método: GET

- Retorno: {"stock": integer, "sku": integer}.

Función: Enviar orden de compra

Esta función tiene como objetivo el envío de una orden de compra a nuestra empresa,
recibe como parametro el id de la orden de compra que desean enviarnos
y se retorna un JSON con nuestra respuesta a la orden de compra
y el id de la orden.

URL: integra1.ing.puc.cl/api/oc/recibir/:idoc

Método: GET

Retorno: {"aceptado": boolean, "idoc":string}

Función: Enviar factura

Esta función tiene como objetivo el enviar una factura a nuestra empresa,
recibe como para parametro el id de la factura y retorna un JSON con su validacion
de nuestra empresa, si es que la factura corresponde a lo que se desea comprar,
y el id de la factura.

URL: integra1.ing.puc.cl/api/facturas/recibir/:idfactura

Método: GET

Retorno: {"validado": boolean, "idfactura": string}

Función: Enviar transacción

Esta función tiene como objetivo el enviar una transacción a nuestra empresa,
recibe como parametro el id de la transacción y el id de la factura que se desea pagar.
Retorna un JSON con la validación de nuestra empresa, que el pago este bien hecho
y corresponda al monto pactado en la factura, y el id de la transacción.

URL: integra1.ing.puc.cl/api/pagos/recibir/:idtrx/:idfactura

Método: GET

Retorno: {"validado": boolean, "idtrx": string}

Función: Recibir despacho

Un cliente la llama cuando realiza un despacho a nuestra empresa
y envía como parametro el id de la factura, para confirmar que el despacho está recibido

URL: integra1.ing.puc.cl/api/despachos/recibir/:idfactura

Método: GET

Retorno: {"validado": boolean}
